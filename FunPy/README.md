# FunPy

Fun Python mood booster!  

**What it does:**
- Pops up funny messages when you save Python files.
- Encourages you with jokes, tips, and playful feedback.

**Commands:**
- `FunPy: Check Python Mood` – manually trigger a message.

**Installation:**
1. Press F5 in VS Code to run in Extension Development Host.
2. Open a Python file, save it, and enjoy the messages!
